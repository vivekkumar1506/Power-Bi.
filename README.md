# 📊 Sales Dashboard 2024 (Power BI Project)

## 📌 Overview

This project is an interactive **Sales Dashboard** built using Power BI.
It provides insights into sales performance, profit analysis, and regional distribution using a dataset of 5000 records.

The dashboard helps users quickly understand business trends and make data-driven decisions.

---

## 🚀 Key Features

* 📈 Total Sales and Total Profit KPIs
* 📊 Sales distribution by Region and Category
* 📉 Profit analysis across different categories
* 🧭 Interactive filters (Region slicer)
* 📅 Time-based trend analysis
* 🎯 Clean and modern UI design

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* Microsoft Excel (Data Source)
* DAX (Data Analysis Expressions)

---

## 📂 Dataset Details

The dataset contains 5000 rows with the following columns:

* Date
* Product
* Category
* Region
* Sales
* Profit

---

## 📊 Dashboard Components

### 🔹 KPI Cards

* Total Sales: 153M
* Total Profit: 19M

### 🔹 Charts Used

* Bar Chart → Sales by Region
* Pie Chart → Sales by Category
* Line Chart → Profit Trend
* Donut/Gauge → Product Count

### 🔹 Filters

* Region slicer for dynamic filtering

---

## 🧠 DAX Measures Used

```DAX
Profit % = DIVIDE(SUM(Profit), SUM(Sales), 0)
```

---

## 🎨 Design Highlights

* Gradient background for modern UI
* Consistent color theme
* Clean layout with proper spacing
* Easy-to-read typography

---

## 📈 Insights from Dashboard

* Electronics category generates highest revenue
* North and East regions have maximum sales
* Accessories category has lower profit contribution
* Profit trend shows category-based variation

---

## 💡 Use Cases

* Business performance tracking
* Sales analysis
* Decision-making support
* Portfolio project for Data Analyst roles

---

## 🔗 How to Use

1. Open Power BI Desktop
2. Load the dataset (Excel file)
3. Explore visuals and filters
4. Interact with slicers for insights
